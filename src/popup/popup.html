Seja bem vindo ao PullBot

Somos uma extensão para google chrome que tem por objetivo ajudar a análise de métricas de desenvolvimento de software de uma equipe, que utiliza o GitHub como plataforma versionamento online.

Para visualizar os gráficos de desenvolvimento basta acessar o GitHub, entrar na milestone desejada e clicar em algum dos botões abaixo.

Para contribuções, reclamações ou esclarecimento de dúvidas, acesse no repositório clicando aqui.
